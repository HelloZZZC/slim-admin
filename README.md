### 脚手架目录
```
.
├── api // 定义接口
├── biz // 业务接口
├── config // 配置项
├── const // 定义常量
├── enum // 定义枚举
├── error // 自定义业务异常
├── global // 全局变量定义
├── kernal // 应用内核
├── middleware // 自定义中间
├── model // 定义模型
├── repository // 模型仓储
├── router // 定义路由
└── util // 工具
```
